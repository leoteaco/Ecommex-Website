## B. Notas para el día del shoot

| Tema                  | Recomendación                                                                                                                                                                              |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Estilo visual         | Cinematográfico, no "stock photo". Sombras marcadas OK, no over-iluminado. Mood profesional pero humano.                                                                                   |
| Paleta                | Identificar oportunidades para que aparezca el teal #1BBCB0 (uniformes, etiquetas, señalización, accesorios). Evitar colores que choquen con navy #2C3E5A.                                 |
| Encuadres             | Capturar **horizontal Y vertical** de cada toma crítica (web + mobile + stories). Pedir tomas con espacio negativo a la izquierda/derecha para overlays de texto.                          |
| Personas              | Que tengan **release firmado** todos los que aparezcan reconociblemente. Mezcla de géneros, edades, roles.                                                                                 |
| Producto en cámara    | Si vas a mostrar producto de clientes reales, pide permiso o usa productos "genéricos". Evita logos de marcas que no sean tus clientes.                                                    |
| Branding al shoot     | Llevar: cinta con logo, cajas con etiqueta Ecommex, uniformes, lanyards, gafetes, plumas, libretas — para que aparezcan orgánicamente en tomas de detalle.                                 |
| Formato de entrega    | RAW + JPG alta calidad, mínimo 3000px en lado largo. Crops en 16:9, 4:5, 1:1, 9:16.                                                                                                        |
| Lentes recomendados   | 24–70mm f/2.8 (todoterreno) + 16–35mm wide para racks profundos + 50mm f/1.8 para retratos. Macro 100mm si hay presupuesto, para detalles tipo S2/P4/D3.                                  |
| Iluminación           | Reforzar zonas oscuras del almacén con LED portátil. Exteriores en golden hour (amanecer/atardecer). Interiores: media mañana con luz lateral natural.                                     |
| Drone                 | Verificar normativa SCT/SEDENA antes. Si no se puede, simular vista elevada desde mezzanine o segundo nivel.                                                                               |
| Día y hora            | Operación pico para tomas con movimiento (mañana/media mañana). Hora dorada para exteriores y aéreas.                                                                                      |
| Preparación previa    | Pasillos despejados, limpiar marcas del piso, inventario presentable a la vista, retirar cajas dañadas o desordenadas. Coordinar con operaciones para que el almacén luzca "su mejor día". |
| Cliente real opcional | Si algún cliente importante puede visitar ese día, capturar un meeting real → oro para case studies.                                                                                       |
| Cantidad mínima       | ~80–120 fotos finales editadas. RAW de respaldo: ~600+.                                                                                                                                    |

## D. Checklist del día del shoot

- [ ] Confirmar release de imagen firmados de todos los empleados que aparecerán
- [ ] Coordinar con operaciones para tener almacén "presentable" (orden + limpieza)
- [ ] Llevar materiales branded: cinta, cajas, gafetes, lanyards, uniformes limpios
- [ ] Verificar normativa de drone (si aplica)
- [ ] Tener listos 2–3 productos genéricos (sin marca) para tomas tipo V1–V5
- [ ] Confirmar que el WMS dashboard se pueda mostrar con datos reales o anonimizados
- [ ] Planear shoot exterior alrededor del golden hour (~30 min antes del atardecer)
- [ ] Tener una persona del equipo Ecommex como punto de contacto durante todo el shoot
- [ ] Briefing previo de 15 min con el fotógrafo: enseñarle el sitio web actual + paleta + tono
- [ ] Lista impresa de este shot list, con espacio para checkmark de cada toma capturada
